# Reporting Inaccuracies

This project was fully generated by AI and may contain astronomical inaccuracies. We welcome corrections to improve the accuracy of celestial body data.

## üéØ Common Areas of Inaccuracy

### Celestial Body Properties
- **Relative sizes**: May not be precisely to scale
- **Orbital distances**: Scaled for visualization, not scientific accuracy
- **Orbital speeds**: Approximate, may not match real values
- **Orbital periods**: May be rounded or approximate

### Scientific Facts
- **Descriptions**: May contain oversimplifications or errors
- **Discovery information**: Names, dates, or discoverers may be wrong
- **Physical characteristics**: Temperatures, diameters may be inaccurate
- **Unique features**: May miss important details or contain errors

### Data Sources
Information was sourced from general knowledge and may not reflect:
- Latest scientific measurements
- Most accurate orbital parameters
- Recent discoveries or reclassifications
- Precise physical constants

## üìù How to Report

### Creating an Issue

1. **Check existing issues** first to avoid duplicates
2. **Use the template** below for your report
3. **Provide sources** when possible (NASA, ESA, IAU, peer-reviewed papers)
4. **Be specific** about what's wrong and what's correct

### Issue Template

```markdown
**Type of Inaccuracy**: [Orbital Data / Physical Properties / Scientific Fact / Description / Other]

**Object Affected**: [Planet/Moon/Satellite Name]

**Current (Incorrect) Value**:
[Current value in the code]

**Correct Value**:
[The accurate value with source]

**Source(s)**:
[NASA, ESA, IAU, scientific paper, etc.]

**Additional Context**:
[Any additional details that would help fix this]
```
